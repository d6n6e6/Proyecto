Objetivo General

Diseñar y desarrollar una aplicación para la administración de parqueadero donde ingrese cualquier tipo de vehículo.

Objetivos Especificos

• Registrar mediante el sistema de información, el 100% de los ingresos y salidas para optimizar el proceso de gestión de parqueadero.


• Generar una base de datos completa y administrable para guardar todas las transacciones del parqueadero.

• Relacionar el ingreso y salida de los vehículos de los usuarios frecuentes del parqueadero mediante un sistema que sea de fácil uso.


• Permitir la modificación en los datos de usuarios nuevos y antiguos, con el fin de mantener el registro actualizado.

Planteamiento Del Problema

El parqueadero cuenta con un sistema web la cual le presenta algunas falencias como:

• Varios usuarios en la aplicacion mientras hay un sesion abierta.
• Cuando el operador no hace cierre de caja, otro operador puede seguir cobrando en esa misma.
• La aplicacion es solo de escritorio.

Alcance del proyecto

El proyecto pretende crear una aplicación adaptable que refleje el control de entrada y salida en el sistema de informacion web y estara en la capacidad de mostrar la cantidad de transacciones que realice el parqueadero.

Justificación

Se desea desarrollar un sistema de información web para la administración del parqueadero en donde los administradores, trabajadores y clientes de dicho sistema tengan una herramienta que les permita gestionar la entrada y salida de los vehiculos.
